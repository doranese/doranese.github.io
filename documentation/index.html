Doranese Documentation

Doranese is a language intended to use as few latin base characters in a different writing syllabary. The following characters are the eleven characters of the language. Accented sounds do not apply to the final count.

  • a
    • ā
    • &acedil;
    • &adblac;
  • e
  • i
  • o
  • u
  • b
  • d
  • j
  • s
  • n
  • r
